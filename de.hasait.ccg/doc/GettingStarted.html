CommentCodeGen (CCG) - Getting Started
sourceforge project page Overview Introduction Getting Started Eclipse Development Setup
First of all download the latest version of the plugin from the files section of the sourceforge project page and extract into your eclipse program folder.
Start eclipse and right click onto a project you want to enable commentcodegen for and select
CommentCodeGen
>
Add Nature
.
Now open any supported file (currently only .java or .cpp) and insert a comment where you want the generated code to be inserted.
The comment may look like: 
/*
 * [c]<element attribute="value" />[/c]
 */
or 
/*
 * [c]<element attribute="value">[/c]
 * [c]<innerElement />[/c]
 * [c]</element>[/c]
 */
The blocks [c][/c] of a comment are merged - this is to allow comment formatting. May be future versions are more intellegent in parsing comments and their formattings.
The outer element type must match a registered generator - it receives the XML as input and defines what attributes and inner elements are allowed.
